## Configurando el Sistema de Control Central: "Activar Agente IA" — Una Demostración Detallada

El sistema de control centralizado del Agente IA se encuentra en la sección **Gestor de Bots**. Este completo sistema te permite gestionar la actividad del chatbot, configurar el sistema de detección de intenciones, seleccionar la campaña AI preferida asegurando que coincida perfectamente con las necesidades de tu negocio y tus estrategias de soporte al cliente.

En términos simples, esta funcionalidad ofrece un enfoque altamente personalizable e intuitivo para optimizar el rendimiento de tu chatbot, manteniendo interacciones profesionales y un control estricto sobre su comportamiento.


> Una de las mayores ventajas del Agente IA es que unifica en un solo panel todas las decisiones críticas: qué dice tu bot, cuándo lo dice, cómo maneja preguntas complejas y cuándo escalar a un humano.

### Activar Agente IA

Primero, ve al **Gestor de Chatbots**. Dentro de la sección WhatsApp o Facebook Bot, encontrarás la opción **Agente IA**. Actívala para configurar el Agente IA con la campaña de entrenamiento AI previamente configurada.

Una vez activado el Agente IA, todas las campañas de chatbot previamente entrenadas aparecerán aquí. Selecciona tu **Campaña de Entrenamiento AI** previamente creada.

### Activar Detección de Intenciones

Activar la **Detección de Intenciones** permite que tu chatbot identifique las consultas de los usuarios y las empareje con intenciones predefinidas. Al configurar la Detección de Intenciones, creas interacciones individualizadas basadas en los requisitos de tus usuarios.


### Nombres de Campaña

Etiqueta y organiza tus campañas de detección de intenciones para una gestión y monitoreo mucho más sencillos.

### Añadir Frases de Ejemplo

Define un conjunto de frases de ejemplo para cada intención para asegurar el reconocimiento incluso si los usuarios escriben sus consultas con redacción o palabras diferentes.

### Activar Disparador de Flujo Bot

Conecta flujos de bot particulares a cada intención y el chatbot podrá ofrecer respuestas automatizadas o acciones predefinidas basadas en la intención reconocida.

### Activar Asignación de Agente

Enruta consultas críticas o complejas a agentes humanos para soporte personalizado.

### Activar Respuestas Impulsadas por IA

Crea campañas de entrenamiento AI personalizadas para intenciones particulares de tu chatbot. A diferencia de las opciones estándar de bot o asignación de agentes, esto brinda más control y flexibilidad para manejar consultas complejas de usuarios. Crea una campaña de entrenamiento AI para la intención definida cuando esté activada. Luego agrega recursos como entradas de FAQ, enlaces URL o conjuntos de datos personalizados al chatbot. Esto asegura que la IA pueda responder de manera contextualmente relevante y precisa a las consultas de los usuarios.

### Gestión de Etiquetas

Con nuestra **Gestión de Etiquetas** mejorada, tienes la flexibilidad de agregar y eliminar etiquetas cuando se detecta una intención. Esto te permite etiquetar conversaciones automáticamente para una gestión, informes y análisis más sencillos.

**Agregar Etiquetas:** Aplica automáticamente una o más etiquetas a una conversación. Por ejemplo, si un usuario hace una pregunta sobre precios, puedes agregar una etiqueta "Precios", facilitando filtrar y rastrear todas las consultas relacionadas con precios.

**Eliminar Etiquetas:** Elimina automáticamente una etiqueta de una conversación. Esto es útil para limpiar etiquetas antiguas. Por ejemplo, podrías eliminar una etiqueta "Pendiente" una vez que se complete una acción de seguimiento.

### Gestión de Secuencias

Nuestra función de **Gestión de Secuencias** te permite controlar la suscripción de un usuario a una secuencia predefinida directamente desde una intención. Una secuencia es una serie de acciones o comunicaciones automatizadas, como una campaña de goteo de correos electrónicos o una serie de mensajes de seguimiento.

**Suscribir a Secuencia:** Inscribe automáticamente a un usuario en una secuencia específica cuando se detecta una intención. Por ejemplo, si la intención de un usuario es "Interés en Producto", puedes suscribirlo a una serie de correos electrónicos con más información del producto y un enlace para programar una demostración.

**Cancelar Suscripción de Secuencia:** Elimina automáticamente a un usuario de una secuencia. Esto es útil para evitar comunicaciones no deseadas, como cancelar la suscripción de un usuario de una secuencia de "Nutrición de Leads" una vez que se ha convertido en cliente.


> Al agregar estas opciones, obtienes una forma más dinámica y potente de gestionar las interacciones de los usuarios, asegurando que tu chatbot no solo entienda la intención, sino que también tome las acciones correctas para guiar al usuario y optimizar tus procesos.

### Agente IA para Todas las Consultas

Cuando alguien activa esta opción, la IA responderá a **todas** las consultas de los clientes, independientemente de si coinciden con la palabra clave o plantilla del flujo del bot o no. En este caso, el chatbot debe estar altamente optimizado y entrenado con todos los datos y entradas necesarios.


> Usar "IA para Todas las Consultas" requiere que hayas entrenado exhaustivamente a tu chatbot. Sin un entrenamiento completo, las respuestas podrían ser inexactas o inconsistentes. Recomendamos esta opción solo después de haber alimentado al bot con FAQs, URLs, archivos y datos de API.

### IA solo como Fallback

Si el usuario activa esta opción, el bot solo responderá cuando se active la respuesta **"Sin Coincidencia"**. Es decir, cuando no haya coincidencia con los flujos de bot predefinidos anteriormente, la respuesta AI se activará y responderá según los datos de entrenamiento.

### Activar Memoria Contextual

Al activar esta función, el chatbot recordará los contextos conversacionales anteriores para referencias futuras. Además, puedes configurar el número de mensajes que el chatbot retendrá en su memoria. Por ejemplo, "Para las (Número) conversaciones anteriores con el suscriptor" indica que el chatbot memorizará el contexto de un número seleccionado de mensajes.


> **¿Por qué es importante la memoria contextual?** Sin ella, cada interacción comienza desde cero. Con ella, el chatbot recuerda que el usuario ya preguntó por precios, mencionó su ciudad y dijo que busca un plan empresarial. Esto permite respuestas coherentes y una experiencia mucho más fluida.

### Activar o Desactivar "No asignar agente en horario no laboral"

Si tienes un negocio que requiere que se asignen agentes a ciertas intenciones, puedes usar la siguiente función para tu detección de intenciones:

**No Asignar Agente Durante Horario No Laboral**

El resultado es que tu chatbot aceptará consultas de usuarios fuera del horario laboral. Si la consulta del usuario coincide con una intención que requiere un agente (situaciones más matizadas o ayuda personalizada), el chatbot no enrutará la consulta a un agente humano fuera del horario laboral.

En su lugar, el chatbot hará una de las siguientes acciones:

- Proporcionar una respuesta automatizada informando al usuario que el equipo no está disponible en ese momento.
- Recomendar soluciones alternativas, como acceder a una FAQ, dejar un mensaje o programar un seguimiento durante el horario laboral regular.
- Intentar resolver la consulta sin intervención humana utilizando Respuestas Impulsadas por IA (si está activado).

Esto ofrece soporte ininterrumpido al usuario respetando la disponibilidad del agente.


> El horario laboral se puede configurar desde **Gestor de Bots > Configuración > Configuración de Horario Laboral**.

Aquí estableces el horario laboral para asegurarte de que las asignaciones de agentes se realicen solo dentro del horario comercial especificado, optimizando la experiencia del usuario y la gestión de la carga de trabajo del agente.

### Temas Restringidos

La opción de temas restringidos aparece cuando la opción de detección de intenciones no está desactivada. Configurar esta opción ayudará a tu chatbot a entender qué temas debe evitar. Los temas se pueden mencionar en el cuadro de texto proporcionado. Activarlo asegurará que la conversación se mantenga adecuada y controlada con un tono más profesional.

**Respuesta para Palabras Clave Restringidas**

Establece la respuesta para palabras clave restringidas aquí. Cada vez que un cliente ingrese alguna palabra clave restringida durante la conversación, responderá con el mensaje predefinido. Esta respuesta particular podría configurarse para aclarar la restricción y advertir cortésmente al usuario sobre ella.



### Ejemplo de Configuración

```
Temas Restringidos: política, religión, datos personales
Respuesta: "Lo siento, no puedo proporcionar información sobre este tema. ¿Hay algo más en lo que pueda ayudarte?"
```

### Ejemplo de Uso

Usuario: "¿Cuál es tu opinión sobre el partido político X?"
Chatbot: "Lo siento, no puedo proporcionar información sobre este tema. ¿Te gustaría conocer nuestros planes de precios en su lugar?"
```

### Activar Asignación de Conversación a Equipo o Departamento

Puedes activar la **Asignación de Conversación** a un equipo particular (ej. marketing, servicio técnico, ventas) para soporte detallado o retención potencial de clientes.

### Indicador de Escritura

El indicador de escritura proporciona un toque humano a la conversación, lo que tiene muchas más posibilidades de retener un lead y canalizar al cliente a través del medio adecuado. Cuando el chatbot está "escribiendo", el usuario ve los puntos suspensivos típicos de WhatsApp, lo que hace que la interacción se sienta más natural y menos robótica.

### Retardo para Respuesta AI

El **Retardo para Respuesta AI** es una funcionalidad excepcional que toma tiempo antes de generar una respuesta. En términos de múltiples mensajes del cliente, su calidad de retroalimentación inteligente genera una única respuesta concisa que acumula todas las respuestas en una respuesta intelectual.

Uno de los mayores desafíos en la comunicación con chatbots es hacer que las conversaciones se sientan naturales para los clientes. Las respuestas instantáneas a veces pueden revelar que la respuesta es automatizada, lo que puede reducir la confianza o el compromiso del usuario. Aquí es donde la función **Retardo para Respuesta AI** se convierte en un punto de inflexión.


> Al introducir un retardo corto y configurable (de 0 a 60 segundos), E-SMART360 permite que tu IA responda de una manera más similar a la humana. Esta pausa sutil imita el comportamiento de escritura real, haciendo que las conversaciones se sientan más auténticas y cómodas para los clientes, especialmente en escenarios de ventas o soporte donde el toque humano importa.

Pero esta función va más allá de solo mejorar el realismo.

En conversaciones del mundo real, los clientes a menudo envían múltiples mensajes uno tras otro, a veces aclarando sus pensamientos o agregando más detalles. Sin retardo, un chatbot puede responder a cada mensaje por separado, lo que lleva a respuestas fragmentadas y a veces menos precisas.

Con **Retardo para Respuesta AI**, E-SMART360 espera inteligentemente un período corto, recolecta todos los mensajes entrantes y genera una única respuesta bien estructurada. Esto resulta en:

- Respuestas más precisas
- Mejor comprensión de la intención del usuario
- Conversaciones más limpias y menos desordenadas


> En resumen, no se trata solo de retrasar respuestas, sino de ofrecer una comunicación más inteligente, más humana y más efectiva.

## Entrenamiento IA del Chatbot de E-SMART360

El Agente IA puede ser entrenado para realizar interacciones con los clientes de manera más inteligente y efectiva. Permitir el entrenamiento AI significa que tu chatbot puede reconocer y reaccionar mejor a las consultas. Entrena tu chatbot con datos personalizados, FAQs y URLs, Google Sheets y APIs para ofrecer respuestas contextuales y precisas.


### Crear una Campaña de Entrenamiento IA

1. Ve al **Panel de Control > Configuración > Campaña de Entrenamiento IA**.
2. Haz clic en **Crear** para iniciar una nueva campaña.
3. Ingresa un **Nombre de Campaña** y un **Mensaje de Prompt**.
4. Ajusta el prompt predeterminado para definir el rol y tono del bot.
5. Haz clic en **Guardar** para continuar.

### Entrenar el Chatbot con FAQs

1. Haz clic en el botón **+** debajo de la campaña.
2. Elige entre:
   - **Resumen:** Proporciona respuestas ricas y conscientes del contexto, pero consume más tokens.
   - **FAQs:** Eficiente y estructurada para respuestas rápidas.
3. Sube el contenido en el formato requerido y guarda.

### Entrenar el Chatbot con una URL

1. Haz clic en **Nuevo** bajo entrenamiento por URL.
2. Ingresa la **URL de la Campaña**.
3. Elige un **Tipo de Selector** (ID o Clase) basado en la estructura de la página web.
4. Opcionalmente, elimina contenido innecesario como anuncios o encabezados.
5. Haz clic en **Generar FAQ** o **Generar Respuesta Cruda**, luego guarda.

### Entrenar el Chatbot con un Archivo

1. Ve a **Configuración de Archivos** y haz clic en **Nuevo**.
2. Sube un archivo PDF, Word (.doc) o TXT.
3. Elige el modo de procesamiento:
   - **Generar Respuesta Cruda:** Proporciona una respuesta completa y detallada (mayor uso de tokens).
   - **Generar FAQ:** Divide el contenido en FAQs estructuradas (menor uso de tokens).
4. Guarda el archivo y finaliza el entrenamiento.


> Para un entrenamiento óptimo, combina múltiples fuentes: FAQ para preguntas frecuentes, URLs para documentación actualizada y archivos para información detallada de productos. Cuantos más datos de calidad proporciones, más preciso será tu Agente IA.

## Configuración Adicional del Comportamiento del Chatbot AI

### Configuración de Respuesta "Sin Coincidencia"

Si el chatbot no puede emparejar una consulta, debe responder basándose en los datos entrenados.

1. Ve a **Gestor de Bots > Botones de Acción > Sin Coincidencia**.
2. Selecciona **Respuesta AI** y enlaza la campaña AI entrenada.
3. Activa **Respuesta Sin Coincidencia** en **Configuración**.
4. Guarda los cambios.

### Activar Asistente IA

1. Ve a **Gestor de Bots**.
2. Activa **Activar Asistente IA**.
3. Selecciona la campaña deseada.
4. Elige el modo de respuesta:
   - **Asistente IA para Todas las Consultas:** La IA maneja todas las consultas de los clientes.
   - **IA solo como Fallback:** La IA interviene cuando fallan las reglas predefinidas.

## Preguntas Frecuentes


### ¿Qué es el Agente IA de E-SMART360?

El Agente IA de E-SMART360 es un chatbot potente que puede entrenarse con FAQs, URLs, archivos, Google Sheets y APIs. Además, es el sistema de control centralizado para gestionar y optimizar el comportamiento del chatbot. Permite a las empresas controlar las respuestas AI, detectar la intención del usuario, gestionar la memoria contextual y manejar escenarios de fallback, proporcionando una solución de soporte al cliente altamente personalizable e inteligente.

### ¿En qué se diferencia el Agente IA de un chatbot tradicional?

A diferencia de los chatbots tradicionales basados en reglas que siguen un guion rígido y preprogramado, el Agente IA de E-SMART360 utiliza IA avanzada para comprender el lenguaje natural y la intención del usuario. Proporciona respuestas dinámicas y conscientes del contexto, y puede entrenarse con datos comerciales específicos (FAQs, URLs, archivos, Google Sheets, APIs) para ofrecer interacciones precisas, personalizadas y similares a las humanas.

### ¿Cómo mejora el Agente IA el soporte al cliente?

El Agente IA mejora el soporte al cliente habilitando funciones como:
- **Detección de Intenciones:** Identifica automáticamente el objetivo del usuario y lo dirige a la información correcta o al agente humano.
- **Memoria Contextual:** Recuerda partes anteriores de la conversación para una experiencia más fluida y personalizada.
- **Asignaciones Automatizadas:** Asigna automáticamente consultas complejas a un agente en vivo, asegurando que ningún problema crítico se pierda.
- **Disponibilidad 24/7:** Proporciona soporte instantáneo a los clientes durante todo el día, mejorando la satisfacción y reduciendo los tiempos de espera.

### ¿Puede el Agente IA trabajar con un equipo de soporte humano?

Sí, absolutamente. El Agente IA está diseñado para ser un socio de tu equipo de soporte humano. Puede configurarse para actuar como punto de contacto principal, manejando todas las consultas simples y repetitivas, y luego escalar o transferir sin problemas los problemas más complejos a un agente humano, proporcionándoles el historial completo de la conversación.

### ¿Puede entrenarse el Agente IA con datos comerciales específicos?

Sí. Una característica clave es la capacidad de entrenar al Agente IA en una variedad de fuentes, incluyendo Google Sheets, APIs, FAQs, URLs de sitios web y archivos específicos. Este entrenamiento personalizado asegura que las respuestas del chatbot no solo sean precisas, sino que también estén alineadas con la marca y perfectamente adaptadas a la base de conocimiento única de tu negocio y sus políticas de soporte al cliente.

### ¿Qué es la función 'Retardo para Respuesta AI'?

La función de Retardo para Respuesta AI te permite agregar un retardo corto (0–60 segundos) antes de que la IA responda. Esto hace que las conversaciones se sientan más naturales y humanas, en lugar de respuestas automatizadas instantáneas. Además, permite que la IA acumule múltiples mensajes del cliente y genere una respuesta única y coherente.

### ¿Agregar retardo ralentiza el soporte al cliente?

No, mejora la experiencia en lugar de ralentizarla. Un pequeño retardo mejora la claridad, reduce las respuestas innecesarias y hace que las interacciones se sientan más reflexivas, añadiendo un toque humano que genera un mejor compromiso y satisfacción del cliente.

### ¿Qué temas puede manejar el Agente IA?

Puedes configurar **Temas Restringidos** para definir qué temas debe evitar el chatbot. Cuando un usuario menciona un tema restringido, el chatbot responde con un mensaje predefinido que redirige la conversación. Esto es ideal para mantener un tono profesional y evitar discusiones sobre política, religión, datos personales u otros temas sensibles.

**Resultado:** Reducción del 60% en llamadas telefónicas entrantes. Los pacientes obtienen respuestas inmediatas 24/7.

## Apéndice A: Integración con Chatbots de Palabras Clave

Además del Agente IA con inteligencia artificial avanzada, E-SMART360 ofrece la funcionalidad completa de chatbots basados en palabras clave. Esta aproximación es ideal para empresas que desean tener control granular sobre respuestas específicas mientras permiten que la IA maneje el resto.

### ¿Qué es un Chatbot de Palabras Clave?

Un chatbot de palabras clave funciona detectando términos específicos en los mensajes de los usuarios y respondiendo con flujos predefinidos. Es como tener un conjunto de reglas automáticas: "Si el usuario menciona 'precio', mostrar la tabla de precios". Esta funcionalidad es extremadamente útil para:

- Respuestas rápidas a preguntas frecuentes
- Navegación asistida dentro de la conversación
- Activación de flujos condicionales
- Encuestas y recolección de datos estructurados

### Crear un Chatbot por Palabras Clave Paso a Paso


### Acceder al Gestor de Bots

Ingresa a E-SMART360 y ve al **Panel de Control > Gestor de Bots > Respuesta de Bot > Crear**. Aquí es donde comienza todo.

### Nombrar el Chatbot

Asigna un nombre descriptivo que te permita identificar fácilmente el propósito del bot. Por ejemplo: "Bot_Ventas_ProductoX", "Soporte_Tecnico_Nivel1" o "FAQ_Horarios".

### Definir Palabras Clave y Respuestas

En el campo de palabras clave, ingresa los términos que activarán el bot. Puedes usar el operador "OR" para alternativas y "AND" para combinaciones.

**Ejemplos de configuración:**

| Palabras Clave | Tipo | Respuesta del Bot |
|---|---|---|
| precio, cuánto cuesta, valor, tarifa | OR | Nuestros planes comienzan en $29/mes. ¿Quieres ver la tabla completa? |
| horario, atienden, abren, horas | OR | Atendemos de lunes a viernes de 9:00 a 18:00 y sábados de 10:00 a 14:00 |
| gracias, ok, entendido | OR | ¡De nada! ¿Hay algo más en lo que pueda ayudarte? |
| reclamo AND pedido, queja AND envío | AND | Lamento el inconveniente. Voy a escalar tu caso a nuestro equipo de soporte. |

### Configurar Tipo de Respuesta

Elige el formato de respuesta que mejor se adapte:
- **Texto simple:** Mensajes directos y rápidos
- **Imagen:** Para mostrar productos gráficamente
- **Botones interactivos:** Ofrece opciones al usuario
- **Lista dinámica:** Para menús con múltiples opciones
- **Flujo condicional:** Activa otro flujo de bot según la respuesta
- **Solicitud de ubicación:** Para servicios basados en ubicación

### Activar y Probar

Guarda la configuración y activa el bot. Envía mensajes de prueba desde WhatsApp, Facebook Messenger o web chat para verificar que las palabras clave se detectan correctamente.

### Combinar Palabras Clave con Agente IA: El Enfoque Híbrido

La verdadera potencia de E-SMART360 se libera cuando combinas ambos sistemas:

```mermaid

```

**Ventajas del enfoque híbrido:**

1. **Control preciso:** Las respuestas críticas están 100% controladas por keywords
2. **Cobertura total:** La IA captura cualquier consulta que no tenga keyword asociada
3. **Transición gradual:** Puedes empezar con keywords y migrar a IA progresivamente
4. **Menos entrenamiento:** No necesitas entrenar la IA para todo desde el día uno
5. **Mejor experiencia:** El usuario recibe respuestas precisas para lo común e inteligentes para lo inesperado

## Apéndice B: Automatización de Seguimiento Inteligente con el Agente IA

Una de las funcionalidades más potentes del Agente IA de E-SMART360 es la capacidad de crear flujos de seguimiento automatizados. Estos flujos permiten que el chatbot interactúe con los usuarios de manera proactiva, no solo reactiva.

### ¿Qué son los Seguimientos Automáticos?

Los seguimientos automáticos son secuencias de mensajes que se activan basándose en el comportamiento del usuario o en la detección de intenciones específicas. Por ejemplo:

- Un usuario pregunta por un producto pero no compra → el chatbot envía un recordatorio 24 horas después
- Un usuario abandona el carrito de compras → se activa una secuencia de recuperación
- Un usuario confirma interés → se programa una llamada de seguimiento con ventas

### Configurar un Seguimiento Automático


### Definir el Disparador

El seguimiento puede activarse por:
- **Intención detectada:** Cuando el usuario muestra interés en un producto
- **Tiempo transcurrido:** Por ejemplo, 24 horas después de la última interacción
- **Evento específico:** Como un abandono de carrito o una compra completada
- **Etiqueta asignada:** Cuando se agrega una etiqueta como "Lead Caliente"

### Diseñar la Secuencia de Mensajes

Crea una serie de mensajes automatizados:
- **Mensaje 1 (Día 1):** "Hola [Nombre], notamos que te interesó nuestro producto X. ¿Te gustaría recibir más información?"
- **Mensaje 2 (Día 3):** "Como cortesía, te ofrecemos un 10% de descuento en tu primera compra. Usa el código BIENVENIDO10"
- **Mensaje 3 (Día 7):** "¿Ya probaste nuestro producto? Nos encantaría conocer tu opinión. ¿Puedes responder esta breve encuesta?"

### Configurar Reglas de Salida

Define cuándo detener la secuencia:
- El usuario compra el producto
- El usuario solicita no recibir más mensajes
- El usuario responde negativamente
- Pasa un número máximo de días sin interacción

### Activar y Monitorear

Guarda la configuración y activa el seguimiento. Monitorea las métricas de conversión: tasa de apertura, tasa de clics, tasa de conversión y tasa de bajas.

### Ejemplo de Seguimiento para un E-commerce

Supongamos que tienes una tienda online de ropa:

1. **Usuario visita el catálogo y pregunta por un vestido rojo**
2. **El Agente IA detecta la intención "Interés en Producto"**
3. **El chatbot responde** con fotos, precios y tallas disponibles
4. **Si el usuario no compra**, se activa una secuencia de seguimiento:
   - **Día 1:** "¿Te gustó el vestido rojo? ¡Tenemos envío gratis hoy!"
   - **Día 3:** "El vestido rojo que viste ahora tiene 15% de descuento"
   - **Día 5:** "Últimos días: 20% de descuento en toda la colección. ¡No te lo pierdas!"
5. **Si el usuario compra**, se detiene la secuencia y se envía un mensaje de agradecimiento


> Los seguimientos automatizados pueden aumentar las tasas de conversión hasta en un 30%. La clave está en personalizar los mensajes y respetar los tiempos de espera entre cada comunicación.

## Apéndice C: Estrategias Avanzadas con el Agente IA

### Estrategia 1: Segmentación Dinámica de Clientes

Usando la detección de intenciones y la gestión de etiquetas, puedes segmentar automáticamente a tus clientes:



### Segmentación por Intención

**Ejemplo:** Clientes que preguntan por "precios" reciben etiqueta "Lead_Ventas". Clientes que preguntan por "soporte" reciben etiqueta "Soporte_Técnico".

**Automatización:** Los leads de ventas se asignan automáticamente al equipo de ventas. Los de soporte van al equipo técnico.

### Segmentación por Comportamiento

**Ejemplo:** Clientes que preguntan 3+ veces en un día reciben etiqueta "Cliente_Frecuente". Clientes que no interactúan en 30 días reciben "Inactivo".

**Automatización:** Los clientes frecuentes reciben ofertas exclusivas. Los inactivos reciben campañas de reactivación.

### Estrategia 2: Embudos de Venta Automatizados

Configura un embudo completo de ventas usando el Agente IA:

1. **Atracción:** El chatbot recibe al usuario con un mensaje de bienvenida personalizado
2. **Calificación:** El Agente IA hace preguntas para entender las necesidades del usuario
3. **Presentación:** Basado en las respuestas, el chatbot presenta los productos o servicios más relevantes
4. **Objeción:** La IA responde objeciones comunes usando la base de conocimiento entrenada
5. **Cierre:** Si el usuario muestra intención de compra, se asigna al equipo de ventas
6. **Postventa:** Después de la compra, se activa una secuencia de seguimiento

### Estrategia 3: Soporte Técnico Multicapa

Crea un sistema de soporte con múltiples niveles:

- **Nivel 1 (IA):** El Agente IA responde preguntas frecuentes usando FAQs y URLs entrenadas
- **Nivel 2 (IA Avanzada):** Para consultas más complejas, la IA usa el archivo de documentación técnica
- **Nivel 3 (Humano):** Si la IA no puede resolver, escala al equipo de soporte con contexto completo
- **Nivel 4 (Especialista):** Para casos muy específicos, asigna al ingeniero o especialista correspondiente


> Este sistema multicapa asegura que el 80% de las consultas se resuelvan en los niveles 1 y 2, liberando a tu equipo técnico para los casos que realmente requieren intervención humana.

## Apéndice D: Resolución de Problemas Comunes

### Problema 1: El Agente IA no responde correctamente

**Causa posible:** La campaña de entrenamiento no tiene suficientes datos o los datos no son relevantes.

**Solución:**
1. Revisa que la campaña de entrenamiento tenga al menos 20-30 FAQs bien redactadas
2. Verifica que las URLs utilizadas contengan información actualizada y relevante
3. Asegúrate de que los archivos subidos estén en formato compatible (PDF, DOC, TXT)
4. Aumenta la cantidad de frases de ejemplo en la detección de intenciones
5. Prueba con preguntas reales de clientes para verificar la precisión

### Problema 2: La detección de intenciones no funciona

**Causa posible:** Las frases de ejemplo no son suficientemente diversas o cubren pocos casos.

**Solución:**
1. Agrega al menos 10-15 frases de ejemplo por intención
2. Incluye variaciones: preguntas directas, indirectas, con errores ortográficos, coloquiales
3. Asegúrate de que las frases de ejemplo sean mutuamente excluyentes entre intenciones
4. Activa primero "IA para Todas las Consultas" para ver cómo responde antes de limitar con intenciones

### Problema 3: El chatbot no recuerda conversaciones anteriores

**Causa posible:** La memoria contextual no está activada o está configurada con muy pocos mensajes.

**Solución:**
1. Verifica que la opción "Memoria Contextual" esté activada
2. Aumenta el número de mensajes a recordar (recomendado: 8-10)
3. Verifica que la campaña de entrenamiento soporte memoria contextual
4. Reinicia el chatbot después de cambiar la configuración

### Problema 4: Las asignaciones a agentes humanos no funcionan fuera del horario laboral

**Causa posible:** La opción "No asignar agente en horario no laboral" está desactivada cuando debería estar activada, o viceversa.

**Solución:**
1. Decide si quieres que se asignen agentes fuera del horario laboral
2. Si NO quieres asignaciones fuera de horario → activa "No asignar agente en horario no laboral"
3. Configura correctamente el horario laboral en Gestor de Bots > Configuración
4. Verifica la zona horaria seleccionada

## Preguntas Frecuentes Adicionales


### ¿Qué pasa si un usuario escribe con errores ortográficos?

El Agente IA de E-SMART360 está entrenado para manejar errores ortográficos menores, variaciones gramaticales y expresiones coloquiales. La detección de intenciones no requiere que el usuario escriba exactamente las frases de ejemplo; puede reconocer variaciones siempre que el significado sea similar. Para palabras clave, recomendás incluir variaciones comunes como "precio", "precio", "cuanto cuesta", "cuanto vale" para cubrir errores frecuentes.

### ¿Puedo tener múltiples campañas de entrenamiento AI?

Sí, E-SMART360 permite crear múltiples campañas de entrenamiento AI para diferentes propósitos. Por ejemplo, puedes tener una campaña para soporte al cliente, otra para ventas y otra para consultas técnicas. Luego, usando la detección de intenciones, puedes asignar cada campaña a la intención correspondiente y asegurar que la respuesta sea la más relevante para cada tipo de consulta.

### ¿El Agente IA consume muchos tokens de chat?

El consumo de tokens depende del modo de operación y la cantidad de datos de entrenamiento:
- **FAQs:** Es el modo más eficiente. Solo consume tokens para la pregunta y respuesta seleccionada.
- **Resumen:** Consume más tokens porque procesa el contexto completo para generar una respuesta.
- **Respuesta Cruda:** Es el modo de mayor consumo, pero ofrece las respuestas más detalladas.

Para optimizar el consumo, recomendamos usar el modo FAQ para preguntas frecuentes y el modo Resumen solo para consultas complejas que requieren contexto. E-SMART360 ofrece diferentes planes de tokens AI para adaptarse a las necesidades de cada negocio.

### ¿Cómo se calcula el costo del Agente IA?

El costo del Agente IA se basa en los tokens de AI consumidos. Cada interacción con el chatbot consume una cantidad de tokens que depende de:
1. La longitud del mensaje del usuario
2. La cantidad de datos de entrenamiento consultados
3. La complejidad de la respuesta generada
4. El modo de respuesta seleccionado (FAQ, Resumen o Crudo)

E-SMART360 ofrece un desglose detallado del consumo de tokens en el panel de control para que puedas monitorear y optimizar tus costos.

### ¿Puedo exportar las conversaciones del Agente IA?

Sí, E-SMART360 permite exportar todas las conversaciones en formatos CSV y Excel. Puedes filtrar por fecha, intención, agente asignado, etiquetas y más. Esto es útil para análisis de calidad, entrenamiento de modelos, informes de satisfacción y cumplimiento normativo.

### ¿El Agente IA cumple con regulaciones de protección de datos?

Sí, E-SMART360 cumple con GDPR, CCPA y otras regulaciones de protección de datos. Puedes configurar temas restringidos para evitar que el chatbot maneje información sensible. Además, las conversaciones se almacenan de forma segura con cifrado en reposo y en tránsito. Los usuarios pueden solicitar la eliminación de sus datos en cualquier momento.

### ¿Qué canales soporta el Agente IA?

El Agente IA funciona en todos los canales que soporta E-SMART360:
- **WhatsApp:** A través de la API oficial de WhatsApp Business
- **Facebook Messenger:** Integración directa con páginas de Facebook
- **Instagram DM:** Mensajes directos de Instagram
- **Telegram:** Bots de Telegram personalizados
- **Web Chat:** Widget de chat para sitios web
- **Sitios Web:** Chat en vivo integrado

La configuración del Agente IA se realiza una vez y se aplica a todos los canales seleccionados, manteniendo la consistencia en las respuestas.

### ¿Puedo probar el Agente IA antes de configurarlo completamente?

Sí, E-SMART360 ofrece un modo de prueba donde puedes simular conversaciones con el Agente IA usando diferentes mensajes de prueba. También puedes crear una campaña de entrenamiento con datos de muestra para ver cómo responde antes de subir tus datos reales. Una vez satisfecho con los resultados, puedes activar el Agente IA para tus canales en producción.
