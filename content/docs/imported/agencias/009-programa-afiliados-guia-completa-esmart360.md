## Seguimiento y Atribución de Referidos

E-SMART360 utiliza el método **"Last Click Wins"** para la atribución de referidos. Esto significa:

1. Cada vez que un visitante hace clic en tu enlace de afiliado, se almacena una cookie en su navegador.
2. Esa cookie tiene una validez de **1 año completo**.
3. Si el visitante se registra (incluso si es meses después), el crédito se asigna al **último afiliado** cuyo enlace haya sido clickeado antes del registro.
4. Esto aplica tanto para registros pagos como para usuarios gratuitos que posteriormente se conviertan a un plan de pago.



### Escenario típico

Un usuario hace clic en tu enlace de afiliado → se registra gratis → usa la plataforma por 2 meses → se actualiza a premium.

    **Tú ganas:** El crédito va al último clic (Afiliado B).
  
### Escenario de referido directo

### Costos de Suscripción de E-SMART360

E-SMART360 ofrece varios planes de suscripción para ayudar a las empresas a gestionar sus mensajes en WhatsApp de manera efectiva. Ahora ofrecemos **créditos de mensajería ilimitados** sin límite de uso, lo que facilita la gestión de las necesidades de comunicación.

- **Plan Premium:** Funciones adicionales para adaptarse a diferentes necesidades empresariales. Facturación mensual o anual según tu preferencia.
- **Plan Revendedor:** Para agencias que quieren ofrecer la plataforma como servicio propio con marca blanca.
- **Precios Enterprise:** Para empresas con requisitos específicos, como mayor número de contactos, miembros del equipo o números de WhatsApp.

### Precios de Conversación de WhatsApp Business API

WhatsApp cobra según el **modelo por conversación**, lo que significa que pagas por conversaciones, no por mensajes individuales. Estos cargos se facturan **directamente a Meta (el propietario de WhatsApp)** y no a través de E-SMART360.

A diferencia de otras plataformas, E-SMART360 ofrece un beneficio importante: **cobramos 0% de margen adicional** sobre los cargos de WhatsApp Cloud API, mientras que muchas otras plataformas cobran una comisión adicional del 20-25%.

### Entendiendo los Precios de WhatsApp Cloud API

El modelo de **precios basado en conversaciones de WhatsApp** significa que pagas una tarifa por conversación, que es una ventana de 24 horas donde puedes enviar mensajes ilimitados a un cliente. Estas conversaciones se clasifican según su propósito:

**Categorías de Conversación:**

1. **Conversaciones de Marketing:** Para promociones, anuncios de productos y ofertas.
2. **Conversaciones de Utilidad:** Para actualizaciones relacionadas con pedidos, entregas o estado de la cuenta.
3. **Conversaciones de Autenticación:** Para verificaciones seguras, como contraseñas de un solo uso o recuperación de cuenta.
4. **Conversaciones de Servicio:** Para manejar consultas o soporte al cliente.

**Nota:** Cada tipo de conversación se cobra según el país del cliente y el tipo de mensaje que envías.

### Créditos de Mensajería Ilimitados

Con E-SMART360 obtienes **créditos de mensajería ilimitados**, lo que significa que no hay límite en la cantidad de conversaciones que puedes iniciar o mensajes que puedes enviar. Solo pagas por los **cargos de WhatsApp Cloud API** basados en la tarifa por conversación, que factura Meta.

### Conversaciones Gratuitas

Cada **Cuenta de WhatsApp Business** recibe **1,000 conversaciones de servicio gratuitas** al mes en todos los números de teléfono comerciales. Esto se reinicia al comienzo de cada mes y aplica solo para conversaciones de servicio. Las conversaciones de marketing, utilidad y autenticación **no forman parte del nivel gratuito**.

### Conversaciones de Punto de Entrada Gratuitas

Si un cliente se comunica contigo a través de un **anuncio Click-to-WhatsApp** o una **llamada a la acción de una página de Facebook**, y respondes dentro de las **24 horas**, comienzas una **conversación de punto de entrada gratuita**. Esta conversación gratuita dura **72 horas**, durante las cuales puedes enviar cualquier tipo de mensaje al cliente sin costo adicional.

### Cómo Funciona el Precio Basado en Conversaciones

**Apertura de Conversaciones:**

- **Conversaciones de Marketing, Utilidad y Autenticación:** Cuando envías un mensaje plantilla en una de estas categorías, se abre una ventana de conversación de 24 horas. Puedes enviar múltiples mensajes dentro de esta ventana sin incurrir en cargos adicionales.
- Si ya hay una conversación abierta de la misma categoría, no se abrirá una nueva.
- **Conversaciones de Servicio:** Cuando un cliente te envía un mensaje, se abre una **ventana de servicio al cliente** por 24 horas. Durante este tiempo, puedes enviar mensajes que no sean plantillas sin abrir una nueva conversación.

## Preguntas Frecuentes sobre Estrategia


### ¿Cuánto tiempo tarda en generarse mi primera comisión?

El tiempo varía según tu estrategia. Algunos afiliados generan su primera comisión en los primeros días si ya tienen una audiencia activa. Otros pueden tardar semanas o meses mientras construyen contenido y atraen tráfico. Lo importante es la consistencia: mientras más contenido de valor publiques, más probabilidades tendrás de generar conversiones.

### ¿Puedo usar el logo y la marca de E-SMART360 en mi sitio?

Sí, siempre que sea para promocionar el programa de afiliados de manera legítima. Te proporcionamos materiales promocionales oficiales que incluyen logos, banners e imágenes. No debes modificar estos materiales ni usarlos de manera que puedan engañar a los usuarios.

### ¿Hay algún contenido prohibido para promocionar?

Sí. No está permitido:
  
  - Usar anuncios con palabras clave de marca registrada de E-SMART360 en Google Ads sin autorización
  - Hacer afirmaciones falsas o exageradas sobre la plataforma
  - Promocionar en sitios con contenido ilegal o inapropiado
  - Usar técnicas de spam para promocionar tu enlace
  - Suplantar a E-SMART360 o hacerte pasar por un empleado oficial

### ¿Puedo combinar el programa de afiliados con otros programas de socios?

Sí, siempre que no haya conflicto de intereses. Por ejemplo, puedes ser afiliado y también usar E-SMART360 para tu propio negocio. Si además eres revendedor, consulta con nuestro equipo para entender cómo se gestionan ambos programas.

### ¿Qué hago si un referido cancela su suscripción?

Si un cliente que referiste cancela su suscripción, tu comisión por ese cliente se detiene. No hay penalización para ti. El objetivo es mantener a tus referidos satisfechos para que sigan siendo clientes activos y tú sigas generando ingresos. Por eso es importante recomendar el plan adecuado para las necesidades de cada usuario.

### ¿Hay herramientas de automatización para afiliados?

Actualmente ofrecemos un panel de control completo con estadísticas en tiempo real. No tenemos integración directa con plataformas de automatización de marketing para afiliados, pero puedes usar herramientas como Zapier para conectar tu panel con otras aplicaciones y automatizar notificaciones.

## Glosario de Términos para Afiliados

Para que puedas comunicarte con confianza con tus referidos, aquí tienes un glosario de términos clave:

| Término | Definición |
|---|---|
| **API de WhatsApp Cloud** | La interfaz oficial de Meta para que las empresas envíen y reciban mensajes de WhatsApp de forma programática |
| **Conversación** | Ventana de 24 horas donde puedes enviar mensajes ilimitados a un cliente. Es la unidad de facturación de WhatsApp |
| **Plantilla de mensaje** | Mensaje preaprobado por WhatsApp que se usa para iniciar conversaciones con los clientes (marketing, utilidad, autenticación) |
| **CTA (Call to Action)** | Botón interactivo en un mensaje que lleva al usuario a una acción específica (visitar web, llamar, etc.) |
| **Click-to-WhatsApp** | Anuncio publicitario que al hacer clic abre directamente un chat de WhatsApp |
| **White Label** | Modelo de negocio donde revendes la plataforma con tu propia marca |
| **Cookie de afiliado** | Archivo que se almacena en el navegador del visitante para rastrear que llegó a través de tu enlace |
| **Tasa de conversión** | Porcentaje de visitantes que realizan la acción deseada (registrarse) |
| **ROI (Retorno de Inversión)** | Beneficio obtenido en relación con el tiempo y recursos invertidos en promocionar |
| **Embedded Signup** | Proceso simplificado para conectar WhatsApp Cloud API directamente desde la plataforma |
