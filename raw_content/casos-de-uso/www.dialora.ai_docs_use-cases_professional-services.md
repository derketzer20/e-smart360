# Professional Services

## Law Firms [Permalink for this section](https://www.dialora.ai/docs/use-cases/professional-services\#law-firms)

- Intake Screening → Qualify cases, check statute of limitations
- Court Date Reminders → Call clients before appearances
- Document Collection → List needed documents, track receipt
- Billing Inquiries → Explain charges, payment arrangements
- Case Status Updates → Automated update calls, reduce inbound calls
- Conflict Checks → Gather party names, check against client database
- Referral Routing → Direct to appropriate practice area
- Deposition Scheduling → Coordinate multiple party availability
- Client Satisfaction Surveys → Post-case feedback, online review requests
- Trust Account Notifications → Low balance alerts, replenishment requests

## Accounting Firms [Permalink for this section](https://www.dialora.ai/docs/use-cases/professional-services\#accounting-firms)

- Tax Document Reminders → Call clients for missing W-2s, 1099s
- Appointment Scheduling → Book tax prep sessions
- Extension Filing → Gather info, file automatically
- Quarterly Estimate Reminders → Payment deadline notifications
- Audit Support Scheduling → Coordinate document reviews
- Bookkeeping Check-ins → Monthly receipt collection reminders
- Payroll Processing Confirmation → Verify amounts before processing
- Year-End Planning Calls → Schedule tax strategy sessions
- New Client Onboarding → Collect business info, set up in systems
- IRS Notice Response → Gather details, schedule urgent consultations

## Insurance Agencies [Permalink for this section](https://www.dialora.ai/docs/use-cases/professional-services\#insurance-agencies)

- Quote Generation → Collect info, provide instant estimates
- Policy Renewal Calls → Review coverage, process renewals
- Claims First Notice → Initial claim intake, assign adjusters
- Life Insurance Health Checks → Update beneficiaries, coverage reviews
- Certificate of Insurance Requests → Collect details, email COIs
- Payment Grace Period Alerts → Prevent policy cancellations
- Cross-Sell Campaigns → Identify coverage gaps, offer solutions
- Referral Generation → Ask satisfied customers for referrals
- Risk Assessment Surveys → Business insurance evaluations
- Policy Change Processing → Add vehicles, update addresses
